Brain Neural Network Visualization

Mar 1 2017

This work shows the moving of information in the human brain.

I used a point cloud data of Human brain to construct neural network and define the spread rule of signal.

Used a particle system by compute buffer and vertex shader in Unity 3D.

利用了一个大脑模型的点云数据，利用粒子模拟信息在大脑中网状传播的视觉效果。粒子系统通过 compute buffer 和 vertex shader 完成。
